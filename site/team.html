Наша команда

Команда проекта «Автоматизация формирования индивидуального плана аспиранта» состоит из студентов Московского Политехнического университета, которые работают вместе, чтобы создать решение для упрощения формирования плана аспирантов. Каждый участник вносит уникальный вклад в разработку.

Горячко Дарья

Роль: Анализ ИПА

Дарья отвечает за анализ структуры документа и выявление проблемных мест. Его внимание к деталям помогает улучшить процесс автоматизации.

Амплеенков Даниил

Роль: Интеграция Excel

Даниил интегрирует Excel в процесс, создавая автоматизированные таблицы для ускорения работы с данными.

Жуков Марк

Роль: Шаблоны Word

Марк разрабатывает оптимизированные шаблоны для Microsoft Word, обеспечивая удобство и точность заполнения ИПА.
